# 🔥 Hyperchaotic Finance System — Reproduction & Quantitative Analysis

## 🧭 Overview

This project reproduces and extends a nonlinear dynamical system describing financial markets with **hyperchaotic behavior**.

Based on:

> Yu, H., Cai, G., Li, Y. (2012)
> *Dynamic analysis and control of a new hyperchaotic finance system*
> Nonlinear Dynamics (Springer)

This repository focuses on:

* numerical reproduction
* stability and dynamical analysis
* bifurcation behavior
* chaos control methods
* interpretation from a quantitative finance perspective

---

## ⚡ Key Features

* Chaotic finance system (3D)
* Hyperchaotic extension (4D)
* Lyapunov exponent computation
* Bifurcation analysis (parameters ( c ), ( k ))
* Phase-space visualization
* Chaos control:

  * Speed feedback control
  * Linear feedback control

---

## 🧮 Mathematical Model

This system exhibits **hyperchaos** when multiple Lyapunov exponents are positive.

Key observations:

* nonlinear coupling leads to instability
